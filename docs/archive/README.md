# Archive

This directory holds design, roadmap, and K9-integration docs that are no
longer live authorities. They are kept for history — moved here with
`git mv` so full history is preserved — not maintained going forward.

For current information, use:

- [`docs/reference/memhub-prd.md`](../reference/memhub-prd.md) and its
  addenda in `docs/reference/` — the product authority.
- [`docs/reference/operations.md`](../reference/operations.md) — operational
  detail for every subsystem (retrieval, token accounting, doc ingestion,
  the code index, machine-global memory, cross-machine sync, upgrade/GC).
- `.memhub/rendered/PROJECT.md` (machine-local, generated by `memhub
  render`) — current build state, decisions, and open questions.
